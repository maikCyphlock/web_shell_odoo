# Technology Stack

## Overview

The "Web Shell Odoo" project is built upon a robust and integrated technology stack designed for extending the Odoo ERP system. It leverages Odoo's native capabilities while incorporating modern web technologies for an enhanced user experience.

## Core Technologies

*   **Programming Languages:**
    *   **Python:** The primary backend language, integral to the Odoo framework. Used for business logic, data models, and server-side operations.
    *   **JavaScript:** Used for frontend development, particularly within the Odoo Web Library (OWL) for interactive components.

*   **Frameworks:**
    *   **Odoo Framework:** The foundational ERP framework providing ORM, MVC-like structure, reporting, and core business functionalities.
    *   **OWL (Odoo Web Library):** Odoo's modern frontend framework, built on a component-based architecture, used for developing rich interactive UI elements.

*   **Libraries:**
    *   **Ace Editor:** An embedded code editor provided via CDN, offering advanced features like syntax highlighting, autocompletion, and themes for an optimal coding experience within the web shell.

*   **Database:**
    *   **PostgreSQL:** The default and recommended relational database for Odoo, providing reliable data storage and management.
